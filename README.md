# 🛡️ BIO-METRIC OS
### Training Intelligence & Biometric Monitoring Engine

**Bio-Metric OS** is a high-performance, industrial-grade web interface designed for advanced training protocols and real-time biometric visualization. Built with a focus on "Sovereign" design aesthetics, this system provides a ruthless, efficient environment for tracking physiological performance and workout execution.

---

## 🚀 Key Features

*   **Asynchronous Boot Sequence:** Implements a realistic neural diagnostic and system calibration sequence using JavaScript Promises.
*   **Biometric Pulse Monitor:** A live HTML5 Canvas heartbeat visualizer that simulates physiological exertion based on exercise intensity[cite: 11].
*   **Dynamic Protocol Engine:** Fetches training data from an external JSON database (`Iron.json`) and renders interactive workout blocks on the fly[cite: 11].
*   **Intelligent Rest Management:** Integrated countdown timer with visual progress rings to ensure optimal recovery between sets[cite: 11].
*   **Session Persistence:** Utilizes `localStorage` to save and restore training states, preventing data loss during accidental refreshes[cite: 11].
*   **Data Export:** Features a comprehensive session export tool that generates a timestamped JSON file of all completed metrics[cite: 11].

## 🛠️ Technical Stack

*   **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)[cite: 11]
